# Create: Real Building

**Create: Real Building** is a technical addon for the *Create Mod* (Minecraft 1.21.1) that brings realism to construction. Forget about placing magic floating blocks; this mod introduces a logical lifecycle for concrete, structural reinforcement, and engineering-based building mechanics.

##  Core Features

*   **Realistic Concrete Lifecycle:** Transition from raw limestone to powder, then to liquid, and finally to solid blocks.
*   **Reinforcement Mechanics:** Distinguish between simple concrete and **Reinforced Concrete** by using steel meshes and girders.
*   **Fluid Dynamics:** Different types of liquid cement with varying flow speeds and structural properties.
*   **Industrial Integration:** Fully compatible with Create’s mixers, spouts, and blaze burners.

##  Planned Recipes & Mechanics

### 1. Concrete Powder
Produced in a **Heated Mixer**:
*   `1x Limestone` + `1x Sand` + `1x Clay` ➔ **4x Concrete Powder**

### 2. Liquid Cements
*   **Self-Leveling Cement:** High flow speed, low durability.
    *   `1x Water` + `1x Concrete Powder` ➔ **Self-Leveling Liquid**
*   **Dense Cement:** Slow flow, high resistance.
    *   `1x Water` + `2x Concrete Powder` ➔ **Dense Liquid**
*   **Liquid Concrete (Structural):**
    *   `1x Water` + `4x Cement Powder` + `4x Sand` + `6x Gravel` ➔ **10x Liquid Concrete**

### 3. Reinforcement (The "Armored" Effect)
*   **Casting:** Pouring Liquid Concrete via a *Spout* onto a **Reinforcement Mesh** or **Metal Girder** transforms the state into **Reinforced Concrete**, a block with significantly higher blast resistance and structural integrity.
